Intranet

An Intranet is a private, internal network that is accessible only to an organization's staff. It uses the same technologies as the World Wide Web (like HTTP and HTML), but it is "fenced off" from the public internet by a firewall.

Role in Technology: It serves as a central hub for internal communication, document sharing (HR policies, training manuals), and collaborative tools.
Security: Access is typically restricted to employees through secure logins or by being physically present on the company's local network (LAN)
