Web Server

A Web Server is a computer system (hardware and software) that stores website files and delivers them to users over the internet via the HTTP protocol. It sits in the background and waits for a browser to ask for data. Primary Functions: Hosting: Storing the HTML, images, and CSS files of a website. Hosting: Storing the HTML, images, and CSS files of a website. Examples: Apache HTTP Server, Nginx, and Microsoft IIS
