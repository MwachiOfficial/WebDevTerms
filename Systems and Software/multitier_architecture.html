Multitier Architecture

Also known as N-tier architecture, this design separates an application into logical layers. The most common is the 3-Tier Architecture:

Presentation Tier: The User Interface (HTML/CSS).
Logic Tier: The processing and "brain" of the app (Python, PHP, Java).
Data Tier: The database where info is stored (SQL).

Role: This separation allows developers to update one part of the system without breaking the others.
