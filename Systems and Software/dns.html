Domain Name Server

DNS is a hierarchical and decentralized naming system for computers or resources connected to the Internet. It maintains a directory of domain names and translates them to the numerical Internet Protocol (IP) addresses required for locating and identifying computer services.

Role: It eliminates the need for humans to memorize IP addresses like 172.217.1.14, allowing us to type google.com instead.
