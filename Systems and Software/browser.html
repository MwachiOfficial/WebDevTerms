Web Browser

A Web Browser is a software application designed to retrieve, present, and traverse information resources on the World Wide Web. When a user types a URL, the browser acts as the Client in the network relationship.
Primary Functions:
Requesting: Sending HTTP requests to web servers.
Rendering: Interpreting HTML, CSS, and JavaScript code to display a visual page.
Navigation: Allowing users to move between pages via hyperlinks.
Examples: Google Chrome, Mozilla Firefox, Apple Safari, and Microsoft Edge.
